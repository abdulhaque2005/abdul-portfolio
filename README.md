## 🚀 Overview

I don't just build apps. I architect immersive digital experiences that define the future. This portfolio is built with precision, optimization, and seamless user experiences in mind.

Features include:
- **Cinematic, Interactive UI:** High-performance animations and glassmorphism elements using Framer Motion and modern Tailwind CSS.
- **Embedded AI Assistant:** Meet "AH Assistant"—a custom-built Gemini-powered chatbot trained to answer questions about my skills, projects, and experience.
- **Dynamic Content Sections:** Including Hero, Skills, Projects (with a grid/card UI), and Resume viewing capabilities.
- **Dark/Light Mode Aesthetics:** Carefully crafted color palettes to evoke a premium feel.

## 🛠️ Technology Stack

- **Frontend Core:** React.js, Vite
- **Styling & Animations:** Tailwind CSS, Framer Motion, Lucide React
- **Integration:** Google Gemini API (for the AI Chatbot)

## 📦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm installed on your local machine.

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/abdulhaque2005/abdul-portfolio.git
   ```
2. **Navigate into the directory**
   ```bash
   cd abdul-portfolio
   ```
3. **Install NPM packages**
   ```bash
   npm install
   ```
4. **Setup Environment Variables**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```
5. **Run the Development Server**
   ```bash
   npm run dev
   ```
